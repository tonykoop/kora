# Kora — Learn to Play

Starter songbook for Tony Koop's kora build.

## About the kora

The kora is a 21-string West African harp-lute from the Mande tradition
(Gambia, Senegal, Guinea, Mali). It is played with both thumbs and index
fingers; the other fingers grip the handles on either side of the gourd body.

## Tuning

Standard **"Tomora Bara"** tuning in F major pentatonic. Strings are arranged
in two interlocking rows (left and right), not in linear pitch order.

**Left-hand bridge strings (thumb L + index L):**

| String | Pitch |
|--------|-------|
| L1 | F3 |
| L2 | A3 |
| L3 | C4 |
| L4 | F4 |
| L5 | A4 |
| L6 | C5 |
| L7 | F5 |
| L8 | A5 |
| L9 | C6 |
| L10 | F6 |
| L11 | A6 |

**Right-hand bridge strings (thumb R + index R):**

| String | Pitch |
|--------|-------|
| R1 | G3 |
| R2 | Bb3 |
| R3 | D4 |
| R4 | G4 |
| R5 | Bb4 |
| R6 | D5 |
| R7 | G5 |
| R8 | Bb5 |
| R9 | D6 |
| R10 | G6 |

(Standard Tomora Bara uses F major pentatonic as the tonal centre; the G and
Bb strings add colour tones from the Bb major scale context.)

## Notation

Staff notation in treble clef. No TAB (the kora is not a fretted instrument).
Melodies are notated at sounding pitch.

## Technique

- Strings are plucked outward (away from the gourd) with the pad of the thumb
  or the nail-side of the index finger.
- Alternating thumb and index finger produces the characteristic kora shimmer.
- Louder notes come from more snap speed, not more force — a sharp, quick
  release produces a bell-like tone.
