Kora-Inspired 21-String Prototype

Private engineering build packet for a segmented-bowl hybrid instrument. Cultural and safety review required before public release.

Private-review status: this page is a build-log draft, not a publishable cultural or construction authority.

What This Is

This repo documents a 21-string kora-inspired prototype using a parametric spreadsheet, structured shop files, and conservative validation gates. The goal is to learn from string scaling, bridge loading, resonator behavior, and player ergonomics without overclaiming tradition or acoustic certainty.

The design source is kora-design-table.xlsx. The current packet should stay private until cultural wording, string safety, bridge stability, and neck/tail proof-load checks are complete.

The Design

Known Risks

  • Current treble string schedule is too close to break strength.
  • Bridge/head pressure and bridge rocking require staged testing.
  • Neck and tail anchor must be proof-loaded before full tension.
  • Public language requires cultural review.
